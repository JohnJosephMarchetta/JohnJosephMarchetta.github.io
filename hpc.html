CUDA Quantum Circuit Simulator

MPCS Project
GPU Programming

20-qubit GPU-accelerated simulator with GHZ state preparation and concurrent CUDA stream execution. Nsight profiling revealed that host-to-device (H2D) memory transfers dominated runtime; the CNOT kernel was memory-bound, while the Hadamard gate achieved high SM compute throughput via FP64 arithmetic. Tuned memory layout and kernel launch parameters accordingly for maximum gate-level parallelism.

Lennard-Jones Force Kernel Optimization

MPCS Project
C++ / OpenMP

Achieved up to a 30× speedup of a C++ Lennard-Jones force kernel by restructuring data from Array-of-Structures (AoS) to Structure-of-Arrays (SoA), enabling compiler SIMD auto-vectorization combined with OpenMP parallelism.

Raft Consensus Message Queue

MPCS Project

Distributed message queue exposing a REST API (Flask) for PUT/GET operations on FIFO topic queues, with the full Raft consensus algorithm implemented from scratch and ML training distributed across Docker containers.
