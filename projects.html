CUDA Quantum Circuit Simulator

20-qubit GPU-accelerated simulator with GHZ state preparation and concurrent CUDA stream execution. Used Nsight to identify memory-bound kernels and tune kernel launch parameters for maximum gate-level parallelism.

Distributed message queue in Python exposing a REST API (Flask) for PUT/GET operations on FIFO topic queues. Implemented the full Raft consensus algorithm from scratch, with ML training on Docker containers.
