Projects & Work

Quantum Error Correction Pipeline

MPCS Project
Apache Camel / ActiveMQ

Built a fault-tolerant microservice pipeline that routes and filters error syndromes over JMS, dispatching in parallel to Minimum-Weight Perfect Matching (MWPM), Union–Find, and Majority Vote decoders, with a voting aggregator to reconcile their outputs.

CUDA Quantum Circuit Simulator

MPCS Project
GPU Programming

20-qubit GPU-accelerated statevector simulator with GHZ state preparation and concurrent CUDA stream execution. Used Nsight to identify memory-bound kernels and quantify stream-overlap throughput gains, then tuned memory layout and kernel launch parameters for maximum gate-level parallelism.

Choi Isomorphism

2021
Quantum Information Seminar, Baylor University

Seminar presentation on the Choi–Jamiołkowski isomorphism between quantum channels and states, a foundational tool for reasoning about error correction and channel capacity.

Compile-Time Quantum Circuit Simulator

MPCS Project
C++ Template Metaprogramming

Explored compile-time circuit simulation using C++ template metaprogramming, pushing as much circuit structure as possible into the type system ahead of runtime execution.

Currently Reading & Building Toward

My current focus is on generalized toric codes, higher-group symmetry, and code construction from new mathematical structures — working through bivariate bicycle (BB) code constructions, CSS code formalism, Laurent polynomial stabilizer representations, and anyon condensation at boundaries. This work is aimed at contributing new lattice constructions to the open-source quantum error correction ecosystem.
