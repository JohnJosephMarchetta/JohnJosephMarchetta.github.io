Quantum Error Correction Pipeline

Fault-tolerant microservice pipeline using Apache Camel and ActiveMQ to route and filter error syndromes over JMS. Parallel dispatch to MWPM, Union-Find, and Majority Vote decoders with a voting aggregator.

CUDA Quantum Circuit Simulator

20-qubit GPU-accelerated simulator with GHZ state preparation, concurrent CUDA streams, Nsight profiling, and gate-level parallelism.

High Performance Computing

My HPC work focuses on CUDA, GPU programming, performance profiling, SIMD, OpenMP, compiler optimization, and scientific simulation.

Scientific Computing Optimization

Achieved up to 30× speedup of a C++ Lennard-Jones force kernel by restructuring data from Array-of-Structs to Struct-of-Arrays, enabling SIMD auto-vectorization and OpenMP parallelism.

Compilers

Built a GoLite compiler pipeline in Go: ANTLR lexer/parser, AST, SSA, liveness analysis, LLVM IR, and ARM assembly.
